Comandos Git bash

LS: para saber todas as pastas.
CD + [nome pasta] + /: para entrar em uma pasta.
Git clone + link: para clonar a pasta criada no git hub.
echo laara > index.html: para criar um arquivo.
git status: para saber o estado atual da pasta do git.
git add index.html: para adicionar realmente um arquivo na pasta, ele deixará de ser
git commit -m ("mensagem"): para commitar o arquivo na pasta git.
git push: para enviar o arquivo para o git hub. (se não estiver logado ou se estiver em um pc desconhecido ele solicitará acesso antes de enviar para o git).
